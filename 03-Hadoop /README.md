
# Hadoop

## Overview

Apache Hadoop is an open-source framework designed for distributed storage and distributed processing of large datasets across clusters of commodity hardware.

Hadoop was created to solve Big Data challenges involving massive volumes of structured, semi-structured, and unstructured data.

---

# Why Hadoop?

Traditional systems face limitations when dealing with:

* Massive data volumes
* High-speed data generation
* Diverse data formats
* Scalability requirements
* Cost constraints

Hadoop addresses these challenges through distributed storage and parallel processing.

---

# Key Features

* Distributed Storage (HDFS)
* Distributed Processing (MapReduce)
* Resource Management (YARN)
* Fault Tolerance
* Horizontal Scalability
* Cost Efficiency
* Data Locality

---

# Hadoop Architecture

The Hadoop ecosystem consists of:

## Storage Layer

* HDFS (Hadoop Distributed File System)

## Resource Management Layer

* YARN (Yet Another Resource Negotiator)

## Processing Layer

* MapReduce

## Ecosystem Tools

* Hive
* Sqoop
* Spark
* HBase
* Oozie
* ZooKeeper
* Flume
